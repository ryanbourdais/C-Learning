# Dynamic Array Library

## Goal

Implement a reusable dynamic array for integers.

## Requirements

- Create a struct that tracks:
  - pointer to data
  - current length
  - current capacity
- Implement functions to:
  - initialize
  - append
  - get by index
  - set by index
  - free memory
- Grow the array automatically when it fills up.

## Concepts To Practice

- pointers
- `malloc`
- `realloc`
- `free`
- API design in C

## Stretch Goals

- add insert and remove
- add bounds checking with error codes
- split into `.h` and `.c` files
