# Contact Book

## Goal

Build a simple contact book that stores contacts and can save them to a file.

## Requirements

- Each contact should include:
  - name
  - phone number
  - email
- Add and list contacts.
- Search contacts by name.
- Save contacts to a file and load them when the program starts.

## Concepts To Practice

- structs
- arrays or dynamic allocation
- file persistence
- search logic
- string handling

## Stretch Goals

- edit existing contacts
- delete contacts
- sort contacts alphabetically
