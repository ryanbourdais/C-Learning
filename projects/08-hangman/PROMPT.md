# Hangman

## Goal

Create a terminal version of hangman using words loaded from a file.

## Requirements

- Load candidate words from a text file.
- Pick one word at random.
- Track guessed letters.
- Reveal correct guesses and count wrong guesses.
- End the game on win or loss.

## Concepts To Practice

- strings
- arrays
- file input
- randomization
- state tracking

## Stretch Goals

- add categories
- draw the hangman step by step
- prevent duplicate guesses cleanly
