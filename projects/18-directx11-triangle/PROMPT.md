# DirectX 11: Triangle

## Goal

Render a colored triangle with DirectX 11.

## Requirements

- Build on the previous DirectX 11 setup.
- Compile or load vertex and pixel shaders.
- Create an input layout.
- Upload vertex data.
- Issue a draw call for one triangle.

## Concepts To Practice

- shaders
- input layouts
- vertex buffers
- pipeline state basics
- draw calls

## Evaluation Focus

- How shader compilation and binding compare to OpenGL and Vulkan
- How manageable the API state model feels
- How quickly you can reach a reusable renderer structure

## Notes

- This project is Windows-specific.

## Stretch Goals

- animate triangle position
- add a constant buffer
- move shader code into separate files
