# OpenGL: Textured Sprite

## Goal

Render a textured quad and move it around the window with keyboard input.

## Requirements

- Load an image as a texture.
- Render a quad made from two triangles.
- Pass texture coordinates to the shader.
- Move the quad based on keyboard input.
- Keep the render loop stable and responsive.

## Concepts To Practice

- textures
- vertex attributes
- uniforms
- input handling
- 2D rendering basics

## Evaluation Focus

- How straightforward texture setup feels
- How easy it is to map CPU-side data to shaders
- How much reusable structure starts to emerge

## Stretch Goals

- add simple camera movement
- support sprite scaling
- load shaders from external files
