# Student Gradebook

## Goal

Build a small gradebook that tracks students, grades, and averages.

## Requirements

- Store multiple students.
- Store multiple assignment grades per student.
- Calculate averages.
- Save and load the gradebook from a file.
- Display a summary report.

## Concepts To Practice

- nested structs
- arrays
- aggregation
- file persistence
- program decomposition

## Stretch Goals

- sort by average
- find highest and lowest scores
- support adding assignments dynamically
