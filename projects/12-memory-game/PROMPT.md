# Memory Game

## Goal

Build a terminal card-matching memory game.

## Requirements

- Create a small grid of hidden cards.
- Shuffle pairs randomly.
- Let the user reveal two cards per turn.
- Check for matches.
- End when all pairs are found.

## Concepts To Practice

- arrays
- randomization
- game loops
- state management
- user input

## Stretch Goals

- add move counting
- add a timer
- support different board sizes
