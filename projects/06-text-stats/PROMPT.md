# Text Stats

## Goal

Read a text file and report useful statistics about it.

## Requirements

- Accept a file path as input.
- Count lines.
- Count words.
- Count characters.
- Handle missing or unreadable files gracefully.

## Concepts To Practice

- file I/O
- character-by-character reading
- string and whitespace handling
- error handling

## Stretch Goals

- count paragraphs
- find the longest word
- print the most common character
