# Vulkan: Triangle Pipeline

## Goal

Render a single triangle through a Vulkan graphics pipeline.

## Requirements

- Build on the previous Vulkan setup.
- Create shader modules.
- Create a render pass or equivalent modern rendering setup.
- Create a graphics pipeline.
- Record draw commands for a triangle.
- Render every frame without validation errors.

## Concepts To Practice

- pipeline creation
- shader modules
- render passes
- command recording
- synchronization details

## Evaluation Focus

- How reusable the pipeline setup seems for an engine
- How painful shader and pipeline iteration feels
- Whether the API model matches your preferred engine architecture

## Stretch Goals

- move vertex data into a vertex buffer
- separate device and renderer code
- hot-reload shaders manually
