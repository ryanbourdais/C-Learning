# To-Do List

## Goal

Create a small terminal app to manage a to-do list during a single program run.

## Requirements

- Add tasks
- List tasks
- Mark tasks complete
- Delete tasks
- Store tasks in memory using structs

## Concepts To Practice

- structs
- arrays of structs
- CRUD-style operations
- menu loops
- string handling

## Stretch Goals

- save tasks to a file
- support priorities
- support due dates
