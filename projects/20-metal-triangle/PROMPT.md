# Metal: Triangle

## Goal

Render a colored triangle with Metal.

## Requirements

- Build on the previous Metal setup.
- Create shader functions in Metal Shading Language.
- Create a render pipeline state.
- Upload triangle vertex data.
- Draw the triangle every frame.

## Concepts To Practice

- pipeline state objects
- shader functions
- vertex buffers
- render pass descriptors
- frame encoding

## Evaluation Focus

- How quickly you can get from clear-screen to actual rendering
- How natural the pipeline model feels
- Whether Metal seems viable if Apple support matters to you

## Notes

- This project is Apple-platform-specific.

## Stretch Goals

- add uniform data for transforms
- animate color or position
- move shader code into dedicated files
