# Mini Shell

## Goal

Create a very small shell that reads commands from the user and runs them.

## Requirements

- Show a prompt.
- Read a line of input.
- Run the command.
- Keep looping until the user exits.
- Start with `system()` to keep the first version small.

## Concepts To Practice

- strings
- loops
- parsing input
- working with the operating system

## Stretch Goals

- implement `cd`
- replace `system()` with `fork` and `exec`
- support simple arguments
