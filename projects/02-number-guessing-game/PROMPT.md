# Number Guessing Game

## Goal

Create a terminal game where the program picks a random number and the user keeps guessing until they get it right.

## Requirements

- Generate a random number in a fixed range.
- Prompt the user repeatedly for guesses.
- Tell the user whether the guess is too high or too low.
- Count how many guesses were needed.

## Concepts To Practice

- loops
- conditionals
- `rand` and `srand`
- user input
- simple game flow

## Stretch Goals

- add difficulty levels
- limit the number of guesses
- allow replay without restarting the program
