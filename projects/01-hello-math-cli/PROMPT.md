# Hello Math CLI

## Goal

Build a command-line calculator that supports basic arithmetic operations:

- addition
- subtraction
- multiplication
- division
- modulus

## Requirements

- Accept input either from command-line arguments or interactive terminal input.
- Split the math logic into separate functions.
- Handle invalid operators.
- Prevent division by zero.

## Concepts To Practice

- variables and types
- conditionals
- functions
- command-line arguments
- basic input validation

## Stretch Goals

- support multiple calculations in one run
- support floating-point math
- print a help message
