# OpenGL: Window And Triangle

## Goal

Open a window and render a single colored triangle with OpenGL.

## Requirements

- Create a window and OpenGL context with a helper library such as GLFW or SDL.
- Compile a vertex shader and fragment shader.
- Upload triangle vertex data to the GPU.
- Render the triangle in a loop.
- Exit cleanly and release resources.

## Concepts To Practice

- graphics context setup
- shader compilation
- vertex buffers
- render loops
- basic API debugging

## Evaluation Focus

- How difficult context setup feels
- How clear the shader workflow is
- How much boilerplate is needed to draw one thing

## Stretch Goals

- resize the viewport correctly
- animate the triangle color
- split shader loading into helper functions
