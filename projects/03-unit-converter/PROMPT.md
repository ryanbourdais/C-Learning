# Unit Converter

## Goal

Build a menu-driven terminal program that converts between several units.

## Requirements

- Include at least three conversion categories.
- Example categories:
  - Celsius and Fahrenheit
  - miles and kilometers
  - pounds and kilograms
- Show a simple menu and let the user choose a conversion.
- Validate input and keep running until the user exits.

## Concepts To Practice

- menus
- functions
- numeric input
- loops
- program organization

## Stretch Goals

- add more unit categories
- support both directions for every conversion
- separate menu logic from conversion logic cleanly
